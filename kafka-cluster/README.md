# kafka-cluster

Docker Compose for kafka-cluster

## About

[Kafka](https://kafka.apache.org/) is a distributed event store and stream-processing platform.

## Containers

| Name              | Description                | Port  |
| :---------------- | :------------------------: | ----: |
| kafka-cluster     | kafka-ui port              | 8080  |
|                   | broker-1 port              | 9091  |
|                   | broker-2 port              | 9092  |
|                   | broker-3 port              | 9093  |
